自省

去年年关，恰逢分手。基本每一次有重大的思想或者思想波动的时候，我都会重新下定决心调整一下生活的方向和态度。当时我找小静同志聊天，大意是，我接下来的目标就是安心学习做科研提升自己，并且定下了三个比较明确的目标。分别是关于英语，吉他和技术，期限是一年，如今一年又是快要到头，三项完成的都不过十之二三，都不好意思再跟小静同志说话，实在是羞愧难当。

今天看到一句话，要改变一个人很难，即使是改变，也会很痛苦。我的毛病很多:

  • 生活起伏比较多比较大，导致心态很浮躁了，难以安静下心来做一件事情。
  • 太博爱，不管是对于事物还是对于异性
  • 强大的拖延症
  • 过于理想主义，容易主动蒙蔽自己
  • 又常常表现得不够善良，不接地气

心态浮躁，对任何事情都没有足够的耐心，很难持久地去做一件事情。情绪经常会受到影响，不管是好的坏的，一个不大不小的消息就能让我马上停下手中的活儿。一个并不靠谱的消息就能让我愤愤或者激动半天。一件事情做到中间，经常想法就会产生动摇，所以没有做成过什么让自己可以自豪的事情。

博爱导致很长一段时间感情十分混乱，尤其是单身的时候，容易和女生产生暧昧不清的关系。兴趣爱好上的博爱，导致至今没有发展出来一份拿得出手的东西，写东西不行，吉它不行，打球不太好，等等。技术上博爱，导致从 cisco 路由交换，到 linux 系统编程，脚本编程，再到 Web 开发，到渗透，23 的人了依然没有找到以后将要从事的方向。

拖延症，几乎扼杀了我做好一切事情的可能性。

理想主义，我慢慢觉察到，这个不只是对于金钱社会是坏东西，对自身的良好发展也不是特别有利，至少从某个层面上来说是的。由于理想主义，我丧失了太多思考的可能性。理想主义常常意味着更多的感性催眠，理性思考变少。比如我特别喜欢罗永浩，于是由于罗永浩，我就本能地讨厌方舟子，然而，我真的应该讨厌方舟子么?如果没有喜欢韩寒和罗永浩，我大概不会这么认为。理想主义，蒙蔽了我的双眼。

不够善良，不会说话，不接地气，收获了一些很好的朋友，也得罪不少人。

这些问题中的许多一年前，就已经知道了，却改变得不多，因为这一年来，我没有太大的痛苦。社会主义的心路历程告诉我们，改革确实是有阵痛的，我没有，说明没改太多。

但是，得有阵痛。

  • 集中注意力做事情。这几天练 typing-lessons，充分感觉到每一次的打字出错率以及速度都是与心态完全相关的。一旦感觉到做事效率低下的时候，思考警醒一下自己，又在胡思乱想什么，马上纠正，重新投入到要做的事情中来。
  • 专注地爱。对于恋爱，我还是比较有信心的。一旦投入到恋爱中，无论是情感上还是理智上，我对于恋爱都比做其它事专注真诚多了。在我单身的一段时间里，博爱泛滥。于是某一天，把所有的关系都切断了，准备认认真真地过日子。然后，上帝就安排我邂逅了我现在这份感情，这份我觉得可以一直走下去的感情。从此以后，我的红尘就是她，她叫西西，我喜欢她，她在人群中或许并不出众，却让我喜欢得无法挑剔。所以，人一旦专注认真了，连上帝都会帮你的。除了感情，我目前最需要坚持的就是技术，技术上摒弃大而全的做事风格，专职 Web，并且以 Open Source 的精神来做染缸。
  • 列定可行的计划，找女朋友帮忙监督。慢慢甩掉这个习惯。
  • 任何事情，在做之前进行衡量，做到心中有数。任何观点，在脱口而出之前，想一想，查一查，以辟谣帝的思维方式来进行审视。
  • 或许我需要学习一下普通话，然后学习一下与人交往的艺术。

从另一方面归结一下:

  • 感情上，多和亲人沟通交流，好好地和西西一直走下去。和朋友们做到真诚，辨认请朋友和匆匆过客。
  • 学习上，坚持一样，就是 Web 构建及安全。
  • 学习态度上，踏实，安心，不急不躁。
  • 做事上，不夸大，不好高鹜远，按需学习。
  • 做人上，不求每个人喜欢，但求不太得罪人。
  • 思维上，博纳百家，明辩万理。

一年再过去的时候，不要再自省。
