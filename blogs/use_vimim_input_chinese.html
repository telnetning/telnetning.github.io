使用 vimim 提高 vim 中文输入效率

vim 的一大特点就是在使用过程中，需要在多个状态之间转换。而其命令模式以及普通模式都不支持中文。这就导致中文编辑者在写作的过程中效率较为低。比如为了跳转到上一行，我们需要首先切换到英文输入法，然后跳转，再进入插入模式，最后再切换回中文输入法。

vimim 就是为了解决这个问题而诞生的。vimim 的解决思路是直接在 vim 中内嵌了一个 CJK 输入法，即，在 vim 中输入英文后，可以直接转换到中文。本质是，在 vim 程序层面上，你输入的还是英文，但是 vimim 可以将你输入的英文字母转换成中文。

vimim 的安装和其它的 vim 插件并没有什么区别。你可以在 https://github.com/vim-scripts/VimIM/ 下载到 vimim, 当然你也可以使用 vundle 之类的插件管理插件来安装。

安装完之后，使用 CRTL+6 就可以切换输入法，使用 CRTL+－ 可以开闭输入法。除此之外 vimim 还有一些其它好玩的功能，这里不一一说了。这里主要说一下 vimim 的词库。

vimim 默认是开启云词库的，其云词库包括 sogou、google 等等，虽然是云端的词库，但是响应速度也很快。但是如果我们没有网了呢？云端词库就失效了。为此我们可以使用本地词库。一般使用 vimim.gbk.bsddb 这个海量词库，我们将其下载下来，和vimim.vim放到同级目录，然后，在 .vimrc 中配置：

let g:Vimim_cloud=-1 "关闭vimim云词库

只有关闭了云端词库，本地词库才能生效。而该词库使用了 berkeley db 数据库，并且使用了 python 作为开发开发工具，所以我们需要安装 berkeley db 以及 python 模块 bsddb，在 mac 下我们需要：

brew install berkley-db

sudo su -
YES_I_HAVE_THE_RIGHT_TO_USE_THIS_BERKELEY_DB_VERSION=1 \
BERKELEYDB_DIR=/usr/local/Cellar/berkeley-db/6.1.26 \
pip install bsddb3

这个时候，一切顺利的话，我们已经能使用该本地词库了。但是该词库不一定全，比如，gai 这个 key 的 value 就没有 "该" 这个字。是的，数据库中的数据是以 key-value 的形式存在的。key 就是我们输入的字母，value 就是字母匹配的汉字。为了能够解决数据不完整的问题，以及为了可以自定义词条，我们可以写一个脚本便于像数据库中添加词条。脚本如下：

#!/usr/bin/env python
#-*- coding: utf-8 -*- 
import os, sys, bsddb3

fileName = '/Users/telnetning/.vim/bundle/VimIM/plugin/vimim.gbk.bsddb'

def printError(ErrorType):
    if ErrorType == 1:
        print "Usage: addBsddbEntry.py key value"
    elif ErrorType == 2:
        print "Error: this value is alread exists."
    elif ErrorType == 3:
        print "Error: the db file not exists."
    elif ErrorType == 4:
        print "Error: key must contains letter only."
    else:
        print "Error: ErrorType is Wrong!"

    exit()

def openDb():
    if not os.path.isfile(fileName) or not os.path.exists(fileName) :
        printError(3)
    else:
        return bsddb3.btopen(fileName, 'c')


if __name__ == '__main__':
    if len(sys.argv) != 3:
        printError(1)
        
    if not sys.argv[1].isalpha():
        printError(4)

    db = openDb()
    
    try:
        originEntrys = db[sys.argv[1]]  #str
    except KeyError:
        originEntrys = ''
    
    newValue = unicode(sys.argv[2], 'utf8').encode('gb18030')
    
    for item in str.split(originEntrys):
        if item == newValue:
            printError(2)
            exit()

    newEntrys = newValue if (originEntrys == '') else newValue + ' ' +originEntrys 
    db[sys.argv[1]] = newEntrys

    db.sync()
    db.close()

然后使用该脚本就能添加词条了，比如：

addBsddbEntry.py gai 该

这样就给 gai 这个字母串添加了 "该" 这个字。输入 gai 试试？

而且，可以随便自定义了，比如，给 name 添加一个词条：你自己的名字。你可以按照你自己的想法，随意添加词条。
